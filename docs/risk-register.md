# سجلّ مخاطر بصيرة — Cybersecurity Risk Register

**التاريخ:** 2026-04-22
**الإصدار:** 1.0
**الإطار المرجعي:** `DT-IS-FRM-2320` — إطار إدارة مخاطر الأمن السيبراني (HRSD)
**المراجعة القادمة:** 2026-07-22 (ربع سنوي) أو عند حادثة أمنية أو تغيير معماري جوهري
**المالك:** فريق تطوير بصيرة (مرحلة التطوير) — سيُحوَّل إلى الوكالة عند التسليم

---

## 1. المنهجية

هذا السجلّ يتبع `DT-IS-FRM-2320` §3 في تقييم المخاطر:

1. **تحديد الأصل** → 2. **التهديد** → 3. **الثغرة** → 4. **الاحتمالية (1–5)** → 5. **الأثر (1–5)** → 6. **المخاطرة الجذرية** = احتمالية × أثر → 7. **فعّالية الضوابط الحالية** → 8. **المخاطرة المتبقية** → 9. **خيار المعالجة** (قبول / تخفيف / تحويل / تجنب).

**مقياس الاحتمالية:**
| القيمة | الوصف | التكرار |
|---|---|---|
| 1 | نادرة | مرّة كل 5+ سنوات |
| 2 | بعيدة | مرّة كل 2-5 سنوات |
| 3 | متوسّطة | سنوياً |
| 4 | مرجّحة | كل 1-6 أشهر |
| 5 | متكرّرة | شهرياً |

**مقياس الأثر:**
| القيمة | الأثر المالي | الأثر السمعيّ | الأثر التشغيلي |
|---|---|---|---|
| 1 | < 10k ريال | لا يُذكر | تأخير ساعات |
| 2 | 10k–100k | داخلي | تأخير يوم |
| 3 | 100k–1M | إقليمي | تأخير أسبوع |
| 4 | 1M–10M | وطني | تأخير شهر |
| 5 | > 10M | دولي + CRPD | فشل مؤسسي |

**تصنيف المخاطرة الإجمالية:**
- 🟢 **منخفضة جداً:** 1–4
- 🟢 **منخفضة:** 5–8
- 🟠 **متوسّطة:** 9–14
- 🔴 **مرتفعة:** 15–19
- ⚫ **حرجة:** 20–25 (إبلاغ فوري لـ NCA)
