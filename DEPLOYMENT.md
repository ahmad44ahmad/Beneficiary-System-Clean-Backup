# Deployment Guide

## Deploying to Vercel

This project is optimized for deployment on [Vercel](https://vercel.com). Follow these steps to deploy your application:

### Prerequisites
- A Vercel account.
- The project code pushed to your GitHub repository (already done).

### Steps

1.  **Log in to Vercel**: Go to [vercel.com](https://vercel.com) and log in.
2.  **Add New Project**: Click on "Add New..." and select "Project".
3.  **Import Repository**:
    - Select "Continue with GitHub".
    - Find your repository (`Beneficiary-System-Clean-Backup`) and click "Import".
4.  **Configure Project**:
    - **Framework Preset**: Vercel should automatically detect **Vite**. If not, select it manually.
    - **Root Directory**: Leave as `./`.
    - **Build Command**: `npm run build` (Default).
    - **Output Directory**: `dist` (Default).
    - **Install Command**: `npm install` (Default).
5.  **Environment Variables**:
    - If you are using Firebase, add your Firebase config variables here (e.g., `VITE_FIREBASE_API_KEY`, etc.).
    - For the **Demo Mode**, no environment variables are strictly required as it falls back to mock data, but you can set `VITE_USE_MOCK_DATA=true` to force it if you implemented that flag (currently the app defaults to demo data if no backend is connected).
6.  **Deploy**: Click "Deploy".

### Verification
- Vercel will build the project and assign a domain (e.g., `beneficiary-system.vercel.app`).
- Visit the URL and verify that the **Master View** and **Dashboard** work as expected.

## GitHub Pages Deployment (Recommended)

Since this project uses client-side routing, we use a `git subtree push` strategy to deploy the `dist` folder to the `gh-pages` branch.

### One-Step Deployment
Run this command from your terminal:

```bash
git add -f dist && git commit -m "Deploy artifacts" && git subtree push --prefix dist origin gh-pages && git reset HEAD~1
```

This command does the following:
1.  Forces the `dist` folder (normally ignored) to be staged.
2.  Commits it temporarily.
3.  Pushes that folder to the `gh-pages` branch on GitHub.
4.  Resets your local history to remove the temporary commit, keeping your main branch clean.

1. مقدمة
1.1 أهمية سياسات مكافحة العدوى
تُعد مكافحة ومنع انتشار العدوى داخل مرافق الرعاية الصحية، وبشكل خاص في مراكز التأهيل الشامل التي تقدم رعاية طويلة الأمد لفئات قد تكون أكثر عرضة للإصابة، من أهم الأولويات لضمان سلامة المستفيدين (المقيمين/المرضى)، والعاملين، والزوار. تشكل العدوى المرتبطة بالرعاية الصحية (HAIs) تحديًا كبيرًا، ويمكن أن تؤدي إلى مضاعفات صحية خطيرة، وزيادة مدة الإقامة، وارتفاع التكاليف، وتدهور جودة حياة المستفيدين. إن تطبيق سياسات وإجراءات مكافحة العدوى بفعالية، استنادًا إلى المعايير العلمية وأحدث الأدلة الإرشادية الوطنية والدولية، يساهم بشكل أساسي في:
حماية الأرواح وجودة الحياة: تقليل معدلات الإصابة بالعدوى ومضاعفاتها الخطيرة، وتحسين النتائج الصحية للمستفيدين.
بيئة آمنة وداعمة: توفير بيئة صحية وآمنة ومريحة لجميع المتواجدين في المركز، تعزز عملية التأهيل والتعافي.
جودة الرعاية المقدمة: الارتقاء بمستوى الخدمات المقدمة من خلال بيئة خالية من العدوى قدر الإمكان، مما يعكس التزام المركز بأعلى معايير الجودة.
سلامة العاملين وحمايتهم: حماية العاملين الصحيين ومقدمي الرعاية من مخاطر التعرض المهني للعدوى، وضمان قدرتهم على تقديم الرعاية بأمان.
كفاءة استخدام الموارد: تقليل الهدر المالي والبشري الناتج عن التعامل مع العدوى المكتسبة وعلاجها.
الامتثال للمعايير واللوائح: الوفاء بالمتطلبات والمعايير التي تفرضها وزارة الصحة والجهات التنظيمية الأخرى في المملكة العربية السعودية.
الاستعداد للطوارئ: تعزيز قدرة المركز على الاستجابة السريعة والفعالة لأي تفشي محتمل للأمراض المعدية.
1.2 أهداف الدليل
يهدف هذا الدليل الإجرائي إلى أن يكون المرجع الأساسي، الشامل، والمفصل لجميع العاملين في مركز التأهيل الشامل بالباحة فيما يتعلق بفهم وتطبيق سياسات وإجراءات مكافحة العدوى. وتتلخص أهدافه المحددة في:
توفير إطار عمل واضح وموحد: تقديم شرح تفصيلي ومنظم لجميع السياسات والإجراءات المعتمدة لمكافحة العدوى داخل المركز.
تحديد وتوضيح المسؤوليات: تحديد دقيق لأدوار ومسؤوليات لجنة مكافحة العدوى، وإدارة المركز، وجميع فئات الموظفين (صحيين، إداريين، خدمات مساندة، عمالة شركة التشغيل) في تطبيق ومراقبة هذه السياسات.
ضمان التطبيق المتسق والفعال: تسهيل فهم وتنفيذ الإجراءات الوقائية بشكل صحيح ومتسق في جميع الأقسام والوحدات والممارسات اليومية داخل المركز.
تعزيز ثقافة السلامة والالتزام: رفع مستوى الوعي بأهمية مكافحة العدوى كمسؤولية فردية وجماعية، وترسيخ ثقافة الالتزام بالممارسات الآمنة والسليمة.
دعم برامج التدريب والتقييم: توفير مادة مرجعية أساسية لبرامج التدريب والتوجيه المستمر، وتوفير أداة واضحة لتقييم مدى الالتزام بالسياسات والإجراءات وتحديد مجالات التحسين.
تحقيق الامتثال للمعايير: مساعدة المركز على تحقيق الامتثال التام للمعايير الوطنية والدولية المتعلقة بمكافحة العدوى.
2. الهيكل التنظيمي ولجنة مكافحة العدوى
2.1 تأسيس الهيكل التنظيمي
لضمان تنفيذ فعال ومستدام لبرنامج مكافحة العدوى، يعتمد مركز التأهيل الشامل بالباحة هيكلاً تنظيمياً واضحاً يحدد الأدوار والمسؤوليات ومسارات التواصل. يضمن هذا الهيكل وجود قيادة داعمة وفريق متخصص قادر على الإشراف والتوجيه.
2.2 لجنة مكافحة العدوى بالمركز (التشكيل والمهام)

المهام والمسؤوليات الرئيسية للجنة:
تتولى لجنة مكافحة العدوى بالمركز مجموعة واسعة من المهام والمسؤوليات الحيوية، تشمل على سبيل المثال لا الحصر:
وضع ومراجعة السياسات والإجراءات:
تطوير ومراجعة وتحديث واعتماد جميع السياسات والإجراءات المتعلقة بمكافحة العدوى الخاصة بالمركز.
التأكد من أن السياسات تستند إلى أحدث الأدلة العلمية والإرشادات الوطنية الصادرة عن وزارة الصحة ووزارة الموارد البشرية والتنمية الاجتماعية والهيئات الدولية المرجعية (مثل WHO, CDC).
ضمان أن تكون السياسات قابلة للتطبيق وواضحة لجميع العاملين.
الإشراف والمتابعة والتنفيذ:
الإشراف العام على تطبيق السياسات والإجراءات في كافة أقسام المركز.
متابعة التزام العاملين بالممارسات الصحيحة لمكافحة العدوى.
التأكد من توفر الموارد اللازمة للتنفيذ الفعال.
المراقبة وتحليل البيانات:
مراجعة وتحليل بيانات المراقبة الدورية المتعلقة بمعدلات الالتزام بالممارسات الوقائية (مثل نظافة الأيدي، استخدام PPE).
مراجعة وتحليل بيانات ترصد العدوى المرتبطة بالرعاية الصحية (HAIs) إن كانت تجمع بشكل منهجي.
تحديد الاتجاهات والمشكلات ومؤشرات الخطر المحتملة.
تقييم المخاطر والتخطيط:
إجراء تقييم شامل للمخاطر المتعلقة بالعدوى داخل المركز بشكل سنوي على الأقل.
تحديد الأولويات بناءً على تقييم المخاطر.
وضع خطة عمل سنوية مفصلة لبرنامج مكافحة العدوى تتضمن أهدافًا واضحة وقابلة للقياس وأنشطة محددة وجداول زمنية ومسؤوليات.
التدريب والتوعية:
الموافقة على خطط التدريب والتوعية السنوية والمستمرة.
التأكد من أن برامج التدريب تغطي جميع الفئات المستهدفة وتلبي الاحتياجات الفعلية.
دعم وتشجيع ثقافة التعلم المستمر في مجال مكافحة العدوى.
إدارة التفشيات:
وضع خطة استجابة واضحة للتعامل مع أي تفشي محتمل للعدوى.
قيادة وتنسيق جهود التحقيق الوبائي واتخاذ الإجراءات اللازمة للسيطرة على التفشي والحد من انتشاره.
التواصل والتنسيق:
التنسيق الفعال بين الأقسام المختلفة داخل المركز (الطبية، التمريض، الخدمات المساندة، الإدارة).
التواصل مع الجهات الصحية الخارجية (وزارة الصحة، الطب الوقائي) بشأن الإبلاغ عن الأمراض المعدية، وتلقي التوجيهات، وتطبيق اللوائح الوطنية.
التواصل مع إدارة المركز لضمان الدعم اللازم للبرنامج.
الاجتماعات والتوثيق:
عقد اجتماعات دورية منتظمة (كل ثلاثة أشهر على الأقل، أو بشكل أكثر تكرارًا إذا لزم الأمر).
توثيق دقيق لجميع محاضر الاجتماعات، بما في ذلك الحضور، والمواضيع التي تمت مناقشتها، والقرارات المتخذة، والتوصيات، والمسؤوليات، والجداول الزمنية.
متابعة تنفيذ القرارات والتوصيات الصادرة عن الاجتماعات السابقة.
رفع التقارير الدورية عن أنشطة اللجنة ونتائج المراقبة إلى إدارة المركز والجهات المعنية.
3. نظام المراقبة والتوثيق
السياسة: يُعد إنشاء وتطبيق نظام فعال للمراقبة والتوثيق أمرًا حيويًا لنجاح برنامج مكافحة العدوى. يهدف هذا النظام إلى القياس المستمر للالتزام بالممارسات الوقائية، وتقييم فعالية التدخلات، وتحديد مجالات التحسين، والكشف المبكر عن أي مخاطر أو تفشيات محتملة للعدوى.
الإجراءات:
3.1 أهمية المراقبة والتوثيق
قياس الأداء: توفير بيانات موضوعية حول مدى التزام العاملين بالسياسات والإجراءات.
تحديد المشكلات: المساعدة في تحديد نقاط الضعف في الممارسات أو الأنظمة التي قد تزيد من خطر العدوى.
تقييم الفعالية: قياس تأثير التدخلات والإجراءات الوقائية المطبقة على معدلات الالتزام أو معدلات العدوى (إن أمكن).
الكشف المبكر: رصد أي زيادة غير طبيعية في حالات العدوى أو ظهور أنماط مقاومة للمضادات الحيوية، مما يسمح بالاستجابة السريعة.
التحسين المستمر: توفير أساس لاتخاذ قرارات مستنيرة وتوجيه جهود التحسين المستمر.
المساءلة والامتثال: توفير دليل موثق على تطبيق معايير مكافحة العدوى والامتثال للوائح.
3.2 إجراءات المراقبة
الملاحظة المباشرة:
تعتبر الأداة الأساسية والأكثر فعالية لمراقبة الممارسات الفعلية للعاملين.
يقوم ممارسو مكافحة العدوى والمشرفون المباشرون (مثل مشرفي التمريض) بمراقبة:
نظافة الأيدي: هل يتم تطبيقها في اللحظات الخمس؟ هل تتم بالطريقة الصحيحة والمدة الكافية؟
استخدام PPE: هل يتم اختيار المعدات المناسبة؟ هل يتم ارتداؤها وخلعها بالتسلسل والطريقة الصحيحة؟
تطبيق احتياطات العزل: هل يتم الالتزام بمتطلبات العزل المحددة (لافتات، PPE، أدوات خاصة)؟
التعامل مع الأدوات والنفايات: هل يتم الفرز والتخلص بشكل آمن؟ هل يتم التعامل مع الأدوات الملوثة بشكل صحيح؟
التنظيف والتطهير: هل يتم اتباع الإجراءات الصحيحة واستخدام المطهرات المناسبة؟
يجب أن تتم الملاحظة بشكل دوري وغير معلن قدر الإمكان للحصول على صورة واقعية للممارسات.
قوائم المراجعة (Checklists):
استخدام قوائم مراجعة موحدة ومفصلة لتقييم الالتزام بإجراءات محددة بشكل منهجي.
تساعد قوائم المراجعة على ضمان تغطية جميع النقاط الهامة وتوحيد عملية التقييم.
أمثلة: قائمة مراجعة نظافة الأيدي (تقييم توفر المستلزمات والالتزام)، قائمة مراجعة تطهير البيئة، قائمة مراجعة تطبيق حزمة رعاية، قائمة مراجعة إدارة النفايات.
يتم استخدامها بشكل دوري (يومي، أسبوعي، شهري) حسب أهمية الإجراء وطبيعة القسم.
3.3 جمع البيانات وتحليلها
بيانات الالتزام: جمع وتسجيل نتائج الملاحظات وقوائم المراجعة بشكل منتظم لحساب معدلات الالتزام (مثل معدل الالتزام بنظافة الأيدي).
مؤشرات الأداء (KPIs): جمع بيانات حول مؤشرات الأداء الرئيسية الأخرى المحددة من قبل اللجنة (مثل معدل اكتمال التدريب، عدد حوادث الوخز بالإبر، نسبة العاملين المحصنين ضد الإنفلونزا).
بيانات العدوى (إن أمكن): إذا كان المركز يطبق نظامًا لترصد العدوى المرتبطة بالرعاية الصحية (HAIs)، يتم جمع البيانات حول أنواع العدوى ومعدلاتها ومسبباتها وفقًا للبروتوكولات المعتمدة.
التحليل: تحليل جميع البيانات المجمعة بشكل دوري (شهري أو ربع سنوي) لتحديد:
الاتجاهات (Trends): هل هناك تحسن أم تدهور في الالتزام أو معدلات العدوى؟
الأنماط (Patterns): هل هناك أقسام أو فئات وظيفية أو أوقات معينة يقل فيها الالتزام؟
المشكلات الحادة (Outliers): هل هناك ارتفاع مفاجئ يتطلب تحقيقًا فوريًا؟
مقارنة الأداء (Benchmarking): مقارنة أداء المركز بمؤشرات وطنية أو مرجعية (إن توفرت).
3.4 التوثيق والسجلات المطلوبة
يُعد التوثيق الدقيق والمنظم جزءًا لا يتجزأ من نظام المراقبة، ويجب الاحتفاظ بالسجلات التالية وتحديثها بانتظام:
سجلات التدريب والتوعية: سجلات حضور الموظفين لجميع الدورات التدريبية وورش العمل المتعلقة بمكافحة العدوى، ونتائج تقييمات الكفاءة.
سجلات المراقبة والتدقيق: نتائج الملاحظات المباشرة، قوائم المراجعة المكتملة، تقارير التدقيق الدوري.
محاضر اجتماعات لجنة مكافحة العدوى: توثيق جميع المناقشات والقرارات والتوصيات والإجراءات المتخذة.
سجلات الحوادث والتعرض المهني: تقارير حوادث الوخز بالإبر والتعرض للدم وسوائل الجسم، والإجراءات المتخذة للمتابعة والعلاج.
السجلات الصحية للعاملين: ملفات فردية تتضمن نتائج الفحوصات الطبية، سجلات التطعيم، وأي تقييد للعمل.
سجلات التنظيف والتطهير: جداول وقوائم مراجعة مكتملة توثق أنشطة التنظيف والتطهير الدورية والنهائية.
سجلات إدارة النفايات: سجلات تتبع كميات النفايات المجمعة وتواريخ نقلها والتخلص منها (حسب متطلبات "موان").
سجلات الصيانة (للأجهزة ذات الصلة): سجلات صيانة واختبار أجهزة التعقيم (إن وجدت)، أنظمة التهوية في غرف العزل، محطات غسل العين، إلخ.
تقارير تحليل البيانات: التقارير الدورية التي تلخص نتائج المراقبة وتحليل الاتجاهات ومؤشرات الأداء.
خطط العمل التصحيحية: توثيق خطط العمل الموضوعة لمعالجة أوجه القصور ومتابعة تنفيذها ونتائجها.
4. الاحتياطات القياسية والعمليات الإجرائية (التنفيذ والتطبيق)
السياسة: تمثل الاحتياطات القياسية والإجراءات التفصيلية الواردة في هذا القسم الممارسات الأساسية واليومية التي يجب على جميع العاملين في مركز التأهيل الشامل بالباحة الالتزام بها بدقة وتطبيقها بشكل متسق مع جميع المستفيدين وفي جميع الأوقات، بغض النظر عن حالة العدوى المعروفة أو المشتبه بها للمستفيد، وذلك بهدف منع انتقال العدوى وضمان بيئة آمنة للجميع.
4.1 الاحتياطات القياسية (Standard Precautions)
هي مجموعة من الممارسات الوقائية الأساسية التي تشكل حجر الزاوية في برنامج مكافحة العدوى، وتشمل العناصر التالية التي سيتم تفصيلها:
نظافة وتطهير الأيدي.
استخدام معدات الوقاية الشخصية (بناءً على تقييم المخاطر).
النظافة التنفسية وآداب السعال.
ممارسات الحقن الآمن والتقنية التعقيمية.
التعامل الآمن مع الأدوات والمعدات الملوثة.
التنظيف والتطهير البيئي.
إدارة النفايات الطبية بشكل آمن.
4.2 نظافة وتطهير الأيدي (Hand Hygiene)
الأهمية: الإجراء الأكثر أهمية وفعالية في منع انتقال معظم أنواع العدوى المرتبطة بالرعاية الصحية.
الإجراءات التنفيذية:
4.2.1 الالتزام باللحظات الخمس (WHO 5 Moments): يجب تطبيق نظافة الأيدي فوراً في هذه اللحظات الحاسمة:
قبل ملامسة المستفيد: لحمايته من الجراثيم التي قد يحملها العامل على يديه. (مثال: قبل المساعدة في الحركة، قبل الفحص البدني).
قبل القيام بإجراء نظيف أو يتطلب تعقيمًا: لمنع دخول الجراثيم (حتى جراثيم المستفيد نفسه) إلى جسمه أثناء الإجراء. (مثال: قبل تغيير الضمادات، قبل تحضير أو إعطاء الدواء، قبل إدخال قسطرة).
بعد خطر التعرض لسوائل الجسم: لحماية العامل نفسه وحماية البيئة من التلوث بعد التعامل مع الدم، البول، الإفرازات، أو أي مواد يحتمل تلوثها (حتى لو كان يرتدي القفازات). (مثال: بعد إزالة القسطرة، بعد تنظيف إفرازات، بعد التعامل مع عينات المختبر).
بعد ملامسة المستفيد: لحماية العامل نفسه والبيئة من جراثيم المستفيد. (مثال: بعد المصافحة، بعد المساعدة في النظافة الشخصية، بعد الفحص البدني).
بعد ملامسة محيط المستفيد: لحماية العامل نفسه والبيئة من الجراثيم التي قد تكون على الأسطح والأدوات المحيطة بالمستفيد، حتى لو لم يتم لمس المستفيد مباشرة. (مثال: بعد تعديل جهاز، بعد لمس حاجز السرير، بعد ترتيب طاولة المستفيد).
4.2.2 طريقة غسل اليدين بالماء والصابون (40-60 ثانية):
الدواعي: إلزامية عند اتساخ اليدين بشكل واضح بالتراب أو المواد العضوية، أو بعد استخدام دورة المياه، أو عند التعامل مع حالات يشتبه بإصابتها بعدوى تسببها كائنات مقاومة للمطهرات الكحولية (مثل المطثية العسيرة - C. difficile).
الخطوات:
تبليل اليدين بالماء الجاري.
وضع كمية كافية من الصابون لتغطية جميع أسطح اليدين.
فرك راحتي اليدين ببعضهما البعض.
فرك راحة اليد اليمنى فوق ظهر اليد اليسرى مع تشبيك الأصابع، ثم العكس.
فرك راحتي اليدين مع تشبيك الأصابع.
فرك ظهر أصابع كل يد براحة اليد الأخرى مع قبض الأصابع (تشابك الأصابع).
فرك الإبهام الأيسر بحركة دائرية باستخدام راحة اليد اليمنى، ثم العكس.
فرك أطراف أصابع اليد اليمنى بحركة دائرية ذهابًا وإيابًا في راحة اليد اليسرى، ثم العكس.
شطف اليدين جيدًا تحت الماء الجاري.
تجفيف اليدين تمامًا باستخدام منشفة ورقية ذات استخدام واحد.
استخدام المنشفة الورقية لإغلاق الصنبور (إذا لم يكن يعمل بدون لمس).
4.2.3 طريقة فرك اليدين بالكحول (20-30 ثانية):
الدواعي: هي الطريقة المفضلة في جميع اللحظات الخمس الأخرى عندما لا تكون اليدان متسخة بشكل واضح، وذلك لفعاليتها العالية وسرعتها وسهولة توفرها عند نقطة الرعاية.
الخطوات:
وضع كمية كافية (ملء راحة اليد) من المطهر الكحولي (بتركيز 60-80%) لتغطية جميع أسطح اليدين.
فرك راحتي اليدين ببعضهما البعض.
فرك راحة اليد اليمنى فوق ظهر اليد اليسرى مع تشبيك الأصابع، ثم العكس.
فرك راحتي اليدين مع تشبيك الأصابع.
فرك ظهر أصابع كل يد براحة اليد الأخرى مع قبض الأصابع (تشابك الأصابع).
فرك الإبهام الأيسر بحركة دائرية باستخدام راحة اليد اليمنى، ثم العكس.
فرك أطراف أصابع اليد اليمنى بحركة دائرية ذهابًا وإيابًا في راحة اليد اليسرى، ثم العكس.
الاستمرار في الفرك حتى تجف اليدان تمامًا (لا تقم بتجفيفها بمنشفة أو بالهواء).
4.2.4 متطلبات إضافية لنظافة الأيدي:
العناية بالأظافر: يجب أن تكون الأظافر طبيعية، قصيرة (لا يتجاوز طولها ربع بوصة)، ونظيفة.
منع الممارسات الخاطئة: يمنع منعًا باتًا ارتداء الأظافر الصناعية أو استخدام طلاء الأظافر للعاملين الذين يقدمون رعاية مباشرة للمستفيدين.
المجوهرات: يجب تجنب ارتداء الخواتم (باستثناء خاتم زواج بسيط أملس) والأساور والساعات أثناء تقديم الرعاية.
صحة الجلد: الحفاظ على صحة جلد اليدين باستخدام كريمات مرطبة معتمدة، وتغطية أي جروح أو خدوش بضمادات مقاومة للماء.
توفر المستلزمات: يجب أن تكون أحواض الغسيل مجهزة بالماء الجاري (يفضل التحكم به بدون لمس)، والصابون (العادي والمضاد للميكروبات حسب الحاجة)، والمناشف الورقية. يجب أن تكون موزعات المطهر الكحولي متوفرة بكثرة عند نقاط تقديم الرعاية (بجانب كل سرير، عند مداخل الغرف، في مناطق تحضير الدواء والطعام، على عربات الإجراءات).
4.3 استخدام معدات الوقاية الشخصية (Personal Protective Equipment - PPE)
الأهمية: تشكل خط الدفاع الأخير لحماية العاملين عند احتمال التعرض لمواد معدية، وتستخدم بناءً على تقييم دقيق للمخاطر.
الإجراءات التنفيذية:
4.3.1 أنواع المعدات ودواعي الاستخدام:
القفازات (Gloves):
الدواعي: ارتداؤها عند توقع ملامسة الدم، سوائل الجسم، الإفرازات، الفضلات، الأغشية المخاطية، الجلد غير السليم، أو الأسطح والمعدات الملوثة. إلزامية عند تطبيق احتياطات التلامس.
الأنواع: قفازات فحص نظيفة (غير معقمة) للاستخدام العام. قفازات معقمة للإجراءات الجراحية أو التي تتطلب تعقيمًا صارمًا (مثل إدخال القسطرة المركزية). قفازات متينة (Utility gloves) لأعمال التنظيف والتعامل مع النفايات.
ملاحظات هامة: تغيير القفازات بين المهام على نفس المستفيد إذا انتقلت من منطقة ملوثة إلى نظيفة. تغيير القفازات بين كل مستفيد. خلعها فورًا بعد الانتهاء من المهمة أو عند تمزقها. القفازات لا تغني أبدًا عن نظافة الأيدي. يجب غسل اليدين أو تطهيرها فور خلع القفازات. يمنع غسل القفازات لإعادة استخدامها.
المآزر/الأرواب (Gowns/Aprons):
الدواعي: ارتداؤها لحماية الجلد والملابس عند توقع تناثر أو تلامس واسع النطاق للدم أو سوائل الجسم. إلزامية عند تطبيق احتياطات التلامس.
الأنواع: مآزر عزل نظيفة (غير معقمة) للاستخدام العام في العزل، يفضل أن تكون مقاومة للسوائل وذات أكمام طويلة. أرواب معقمة للإجراءات الجراحية.
ملاحظات هامة: تغيير المئزر بين المستفيدين أو إذا تبلل أو تلوث بشدة.
حماية الجهاز التنفسي (Masks/Respirators):
الكمامة الجراحية:
الدواعي: حماية مرتديها من الرذاذ المتطاير الكبير. حماية الآخرين من إفرازات مرتديها (التحكم في المصدر) - يجب أن يرتديها المريض الذي يعاني من أعراض تنفسية أو يخضع لاحتياطات الرذاذ/الهواء عند الحاجة للخروج من غرفته. إلزامية للعاملين عند تطبيق احتياطات الرذاذ.
ملاحظات هامة: يجب أن تغطي الأنف والفم والذقن بإحكام. تغييرها إذا أصبحت رطبة أو ملوثة أو تالفة، وبين المستفيدين.
كمامة N95 التنفسية (أو أعلى):
الدواعي: حماية مرتديها من استنشاق الجسيمات الدقيقة المحمولة جوًا (الهباء الجوي). إلزامية لجميع الأفراد الذين يدخلون غرفة مريض يخضع لاحتياطات العزل الهوائي (مثل السل، الحصبة، جدري الماء). إلزامية عند إجراء أي من الإجراءات المولدة للرذاذ (AGPs) على مرضى يعانون من أمراض تنفسية معدية (حتى لو كانت تنتقل عادة بالرذاذ).
ملاحظات هامة: تتطلب اختبار ملائمة (Fit-testing) سنويًا على الأقل أو عند حدوث تغيير في شكل الوجه، لتحديد النوع والمقاس المناسب لكل فرد. تتطلب فحص إحكام (User Seal Check) ذاتي في كل مرة يتم ارتداؤها للتأكد من عدم وجود تسرب للهواء. يجب التخلص منها بعد الاستخدام أو إذا أصبحت ملوثة أو تالفة أو مقاومة التنفس زائدة. الاستخدام الممتد أو إعادة الاستخدام المحدود قد يسمح به في حالات النقص الشديد وفقًا لتوجيهات وزارة الصحة الصارمة.
جهاز تنقية الهواء المزود بالطاقة (Powered Air-Purifying Respirator - PAPR):
الدواعي: بديل لكمامة N95 للأشخاص الذين لا يستطيعون تحقيق إحكام جيد (مثل وجود لحية)، أو الذين فشلوا في اختبار الملائمة لجميع الأنواع المتاحة، أو عند الحاجة لحماية مطولة في بيئات عالية الخطورة. يوفر مستوى حماية أعلى.
ملاحظات هامة: يتطلب تدريبًا خاصًا على الاستخدام والتنظيف والصيانة. يجب فحص البطارية وتدفق الهواء قبل كل استخدام. يتم تنظيف وتطهير الأجزاء القابلة لإعادة الاستخدام وفقًا لتعليمات الشركة المصنعة.
حماية العين والوجه (Eye/Face Protection):
الدواعي: ارتداؤها لحماية الأغشية المخاطية للعينين والأنف والفم من تناثر الدم أو سوائل الجسم أو الرذاذ التنفسي. مطلوبة عند إجراءات قد تسبب تناثرًا (مثل الشفط، بعض إجراءات العناية بالجروح) وعند تطبيق احتياطات الرذاذ أو الهواء.
الأنواع: نظارات واقية (Goggles) توفر حماية للعينين. واقي للوجه (Face Shield) يوفر حماية أوسع للوجه.
ملاحظات هامة: يجب أن تكون ملائمة بشكل جيد. يتم تنظيف وتطهير المعدات القابلة لإعادة الاستخدام بين المرضى أو التخلص منها إذا كانت ذات استخدام واحد.
4.3.2 تسلسل الارتداء (Donning): (يجب أن يتم بالترتيب لمنع التلوث)
نظافة الأيدي: غسل اليدين أو تطهيرها بالكحول.
المئزر/الروب: ارتداء المئزر وربطه جيدًا من الخلف (الرقبة والخصر).
الكمامة/كمامة N95: وضع الكمامة على الوجه وتثبيتها بإحكام (الشريط المعدني على الأنف، الأربطة خلف الرأس/الأذنين). إجراء فحص الإحكام لكمامة N95.
واقي العين/الوجه: وضعه على الوجه وتثبيته.
القفازات: ارتداء القفازات بحيث تغطي أطراف أكمام المئزر.
4.3.3 تسلسل الخلع (Doffing): (الأكثر خطورة، يتطلب حذرًا شديدًا لمنع التلوث الذاتي)
القفازات: خلع القفاز الأول بمسكه من الخارج قرب الرسغ وسحبه للخارج ليصبح مقلوبًا. إمساك القفاز المخلوع باليد التي لا تزال ترتدي القفاز. إدخال أصابع اليد العارية تحت حافة القفاز المتبقي عند الرسغ وخلعه ليصبح مقلوبًا فوق القفاز الأول. التخلص منهما في سلة النفايات.
نظافة الأيدي: تطهير اليدين بالكحول.
واقي العين/الوجه: خلعه بمسكه من الأربطة أو القطع الجانبية من الخلف أو الجانب، وتجنب لمس الجزء الأمامي الملوث. وضعه في المكان المخصص لإعادة المعالجة أو التخلص منه.
المئزر/الروب: فك الأربطة (الخصر ثم الرقبة). سحب المئزر بعيدًا عن الرقبة والكتفين بلمس الجزء الداخلي فقط. طي المئزر بحيث يصبح الجزء الخارجي الملوث للداخل ولفه على شكل حزمة. التخلص منه في سلة النفايات.
نظافة الأيدي: تطهير اليدين بالكحول.
الكمامة/كمامة N95: (يتم خلع N95 خارج غرفة العزل الهوائي بعد إغلاق الباب). مسك الأربطة السفلية أولاً ثم العلوية لخلع الكمامة دون لمس الجزء الأمامي. التخلص منها في سلة النفايات.
نظافة الأيدي: غسل اليدين جيدًا بالماء والصابون أو تطهيرها بالكحول كخطوة أخيرة.
4.3.4 التوفر: يجب ضمان توفر جميع أنواع معدات الوقاية الشخصية المطلوبة، بالمقاسات المناسبة، وبكميات كافية في جميع نقاط تقديم الرعاية والمناطق التي قد تتطلب استخدامها. يجب وجود آلية واضحة لطلب وتجديد المخزون.
4.4 احتياطات العزل (Isolation Precautions)
الأهمية: تطبيق إجراءات إضافية محددة لمنع انتقال الممرضات المعروفة أو المشتبه بها بناءً على طريقة انتشارها.
الإجراءات التنفيذية:
التحديد والتصنيف: يجب على الفريق الطبي والتمريضي تحديد الحاجة لتطبيق احتياطات العزل بناءً على التشخيص المؤكد أو الاشتباه السريري والوبائي، وتحديد نوع الاحتياطات المطلوبة (تلامس، رذاذ، هواء، أو مزيج منها).
التطبيق الفوري: تطبيق الاحتياطات المناسبة فور تحديد الحاجة إليها، بالإضافة إلى الاحتياطات القياسية.
متطلبات كل نوع:
احتياطات التلامس (Contact): (اللافتة: خضراء)
الغرفة: غرفة خاصة مفضلة. يمكن تجميع المرضى بنفس الممرض بنفس الغرفة مع الحفاظ على مسافة >1.2 متر وفصل الأدوات.
PPE: ارتداء قفازات ومئزر عند دخول الغرفة أو منطقة المريض.
الأدوات: استخدام أدوات مخصصة للمريض (سماعة، جهاز ضغط) أو تطهير الأدوات المشتركة جيدًا بين كل استخدام.
نظافة الأيدي: ضرورية جدًا قبل وبعد خلع القفازات والمئزر وعند مغادرة الغرفة.
تنظيف البيئة: زيادة وتيرة تنظيف وتطهير الأسطح عالية اللمس.
احتياطات الرذاذ (Droplet): (اللافتة: وردية/حمراء)
الغرفة: غرفة خاصة مفضلة. يمكن التجميع مع الحفاظ على مسافة >1.2 متر.
PPE: ارتداء كمامة جراحية عند العمل على مسافة قريبة (1-2 متر) من المريض. قد يتطلب قفازات ومئزر وحماية عين حسب تقييم خطر التناثر.
المريض: يجب أن يرتدي المريض كمامة جراحية عند الحاجة للخروج من الغرفة.
احتياطات الهواء (Airborne): (اللافتة: زرقاء)
الغرفة: إلزامية غرفة عزل هوائي ذات ضغط سلبي (AIIR) بالمواصفات المعتمدة (ضغط ≥ -2.5 باسكال، ≥ 12 تغيير هواء/ساعة، تصريف للخارج عبر HEPA).
PPE: ارتداء كمامة تنفسية N95 (أو أعلى) تم اختبار ملائمتها من قبل جميع الأشخاص قبل دخول الغرفة. قد يتطلب قفازات ومئزر وحماية عين حسب تقييم خطر التناثر.
الدخول: تقييد الدخول للأشخاص الضروريين فقط. إبقاء الباب مغلقًا دائمًا.
المريض: يجب أن يرتدي المريض كمامة جراحية عند الحاجة القصوى للخروج من الغرفة.
الإجراءات المولدة للرذاذ (AGPs):
عند إجراء أي AGP (مثل الشفط المفتوح، الإنعاش، التنبيب، تنظير القصبات، أخذ مسحة البلعوم الأنفي) على مريض يعاني من عدوى تنفسية (حتى لو كانت تنتقل بالرذاذ عادة مثل كوفيد-19 أو الإنفلونزا):
المكان: يجب إجراؤها في غرفة AIIR إن أمكن، أو على الأقل في غرفة فردية بباب مغلق وتهوية جيدة (مع استخدام جهاز تنقية هواء محمول بفلتر HEPA إن أمكن).
PPE: يجب على جميع المتواجدين ارتداء كمامة N95 (أو PAPR)، حماية للعين (نظارات أو واقي وجه)، قفازات، ومئزر مقاوم للسوائل.
تقليل الأفراد: قصر عدد المتواجدين في الغرفة على الحد الأدنى الضروري.
اللافتات والتعليمات:
وضع لافتة واضحة ومناسبة لنوع العزل (باللون المعتمد وباللغتين العربية والإنجليزية) على باب الغرفة، توضح الاحتياطات المطلوبة و PPE اللازم.
تثقيف الزوار حول الاحتياطات اللازمة وأهمية نظافة الأيدي وارتداء PPE المطلوب قبل السماح لهم بالدخول. يجب توثيق دخول الزوار لغرف العزل الهوائي في سجل خاص.
نقل المستفيدين المعزولين:
تقليل النقل إلى الحد الأدنى الضروري طبيًا.
استخدام بطاقة نقل العزل المخصصة (باللون المناسب) التي توضح نوع العزل والاحتياطات.
إبلاغ القسم المستقبل مسبقًا بنوع العزل لضمان استعدادهم وتوفر PPE.
يجب على المريض ارتداء PPE المناسب (كمامة جراحية للعزل الهوائي/الرذاذ).
تغطية أي جروح أو آفات جلدية للمريض المعزول بالتلامس بضمادات نظيفة.
اختيار مسار النقل الأقل ازدحامًا.
يجب على المرافقين (العاملين) ارتداء PPE المناسب إذا كان هناك احتمال للتعرض أثناء النقل.
4.5 إدارة النفايات الطبية (Medical Waste Management)
الأهمية: التخلص الآمن من النفايات الطبية ضروري لمنع انتشار العدوى وحماية البيئة والصحة العامة.
الإجراءات التنفيذية: (وفقًا للنظام الموحد لدول مجلس التعاون وتعليمات "موان")
4.5.1 التصنيف والفرز عند المصدر:
إلزامي: يجب على الشخص الذي أنتج النفايات فرزها فورًا عند نقطة التولد.
التصنيفات الرئيسية:
نفايات عامة (سوداء): نفايات غير خطرة تشبه النفايات المنزلية (أوراق مكتبية، بقايا طعام غير ملوثة من مناطق غير سريرية).
نفايات معدية (صفراء): أي نفايات يشتبه بتلوثها بمسببات الأمراض أو تقطر دمًا أو سوائل جسم (ضمادات ملوثة، قفازات، مواد مستخدمة في رعاية المرضى المعزولين بأمراض شديدة العدوى حسب السياسة).
نفايات حادة (صفراء - حاوية صلبة): إبر، محاقن، مشارط، أمبولات زجاجية مكسورة، أي أداة يمكن أن تسبب وخزًا أو جرحًا.
نفايات باثولوجية (حمراء): أنسجة، أعضاء، مشيمة (تتطلب معاملة خاصة).
نفايات كيميائية/صيدلانية/سامة للخلايا: تتطلب أوعية وأكياس خاصة (غالبًا بنفسجية للسامة للخلايا) وإجراءات محددة.
4.5.2 الأوعية والحاويات والترميز اللوني:
استخدام الأكياس والحاويات المخصصة والمقاومة للتسرب والتمزق (والثقب للحادة) وبالألوان المعتمدة.
يجب أن تحمل جميع الأوعية علامة الخطر البيولوجي الدولية.
الملء: عدم ملء الأكياس أو الحاويات لأكثر من ثلاثة أرباع (¾) سعتها.
الإغلاق: إغلاق الأكياس والحاويات بإحكام قبل النقل (ربط الأكياس، إغلاق غطاء الحاويات الحادة بشكل دائم).
الملصقات: وضع ملصق واضح على كل كيس/حاوية نفايات خطرة يوضح: اسم المركز، القسم المنتج، نوع النفايات، تاريخ الجمع.
4.5.3 التجميع والنقل الداخلي:
المسؤولية: يقوم عمال النظافة المدربون أو عمال إدارة النفايات بجمع النفايات من الأقسام.
التكرار: يتم الجمع بشكل دوري (يوميًا على الأقل، أو حسب الحاجة وكثافة الإنتاج) وفي أوقات محددة.
PPE: يجب على العمال ارتداء معدات الوقاية الشخصية المناسبة (قفازات متينة، مئزر واقٍ، حذاء آمن، كمامة ونظارات واقية عند الحاجة).
العربات: استخدام عربات مخصصة لنقل النفايات الخطرة، تكون مغلقة، مقاومة للتسرب، سهلة التنظيف، وتحمل علامة الخطر البيولوجي. يجب تنظيفها وتطهيرها يوميًا.
المسارات: استخدام مسارات محددة وأقل ازدحامًا لنقل النفايات.
4.5.4 التخزين المؤقت:
الموقع: تخصيص منطقة (غرفة) آمنة ومغلقة وجيدة التهوية وذات أرضية سهلة التنظيف ومحمية من الآفات لتخزين النفايات الخطرة مؤقتًا قبل نقلها للمعالجة. يجب أن تكون بعيدة عن مناطق المستفيدين والمناطق العامة ومناطق تحضير الطعام.
المدة: يجب ألا تتجاوز مدة التخزين 24 ساعة في الظروف العادية (قد تطول في حال التخزين المبرد لبعض النفايات).
التنظيم: فصل الأنواع المختلفة من النفايات الخطرة داخل منطقة التخزين إن أمكن.
4.5.5 النقل الخارجي والمعالجة والتخلص النهائي:
المسؤولية: يتم التعاقد مع شركة متخصصة ومعتمدة من المركز الوطني لإدارة النفايات ("موان") لنقل ومعالجة والتخلص النهائي من النفايات الطبية الخطرة.
الإجراءات: تتبع الشركة المتعاقدة جميع الأنظمة واللوائح المتعلقة بالنقل الآمن والمعالجة المعتمدة (حرق، تعقيم، إلخ) والتخلص النهائي في المرافق المرخصة.
4.6 التنظيف والتطهير البيئي (Environmental Cleaning & Disinfection)
الأهمية: البيئة النظيفة والمطهرة تقلل من الحمل الميكروبي وتحد من انتقال العدوى غير المباشر.
الإجراءات التنفيذية:
الجداول والبروتوكولات:
وضع وتطبيق جداول وبروتوكولات تنظيف وتطهير مفصلة لجميع مناطق المركز (غرف المستفيدين، مناطق العلاج، المكاتب، الممرات، دورات المياه، المطبخ، المغسلة، إلخ).
تحديد المسؤوليات (عمال النظافة، فريق التمريض لبعض المعدات).
تحديد وتيرة التنظيف (يومي، دوري، نهائي).
التنظيف الروتيني:
تنظيف جميع الأسطح الأفقية والأرضيات يوميًا باستخدام الماء والمنظفات المعتمدة.
التركيز على الأسطح عالية اللمس: تطهير الأسطح التي يكثر لمسها (مقابض الأبواب، مفاتيح الإضاءة، حواجز الأسرة، الطاولات، الهواتف، أجهزة التحكم، أسطح العمل، أزرار المصاعد، مقابض الكراسي المتحركة، أسطح دورات المياه) عدة مرات يوميًا وبعد كل استخدام للمعدات المشتركة، باستخدام مطهر معتمد.
استخدام تقنية التنظيف الصحيحة (من الأعلى للأسفل، من الأنظف للأكثر اتساخًا، تغيير قطع القماش والمماسح بانتظام).
التنظيف النهائي (Terminal Cleaning):
إجراء تنظيف وتطهير شامل وعميق لغرفة المستفيد وجميع محتوياتها بعد خروجه أو إنهاء العزل.
الإشراف: يتم الإشراف على التنظيف النهائي لغرف العزل من قبل الممرضة المسؤولة، وفي حالات التفشي، من قبل ممارس مكافحة العدوى.
تقنيات إضافية (إن وجدت): يمكن استخدام تقنيات تطهير إضافية مثل أجهزة الأشعة فوق البنفسجية (UV) أو بخار بيروكسيد الهيدروجين (HPV) بعد التنظيف اليدوي في حالات خاصة (مثل بعد حالات C. difficile أو تفشيات MDROs) إذا كانت متوفرة ومعتمدة.
المطهرات والمنظفات:
استخدام المنظفات والمطهرات المعتمدة من وزارة الصحة والمتوافقة مع الأسطح والمعدات.
تحضير المحاليل بالتركيز الصحيح وفقًا لتعليمات الشركة المصنعة.
الالتزام بوقت التلامس (Contact Time) المطلوب للمطهر لضمان فعاليته.
الاطلاع على ورقة بيانات سلامة المواد (MSDS) للمطهرات والمواد الكيميائية واتباع احتياطات السلامة عند التعامل معها.
تنظيف وتطهير المعدات الطبية:
المعدات غير الحرجة: تطهير المعدات التي تلامس الجلد السليم (سماعة، جهاز ضغط، ميزان حرارة) بين كل مستفيد.
المعدات الكبيرة: تنظيف وتطهير الأسطح الخارجية للمعدات الكبيرة (أجهزة التنفس، أجهزة الغسيل الكلوي، الحاضنات "إن وجدت") بانتظام وفقًا لتعليمات الشركة المصنعة وسياسة المركز.
معدات العلاج المائي (إن وجدت): يجب تفريغ أحواض العلاج المائي وتنظيفها وتطهيرها بعد استخدام كل مستفيد.
4.6.4 التعامل مع انسكابات الدم وسوائل الجسم (Spill Kit Management):
التوفر: يجب أن تكون مجموعة أدوات التعامل مع الانسكابات البيولوجية (Spill Kit) متوفرة وجاهزة للاستخدام في جميع المناطق التي يحتمل حدوث انسكابات فيها.
المحتويات الأساسية: (قفازات متينة، مئزر مقاوم للسوائل، واقي عين/وجه، كمامة، مادة ماصة/بودرة تجلط، محلول مطهر مناسب مثل الكلور، أكياس نفايات صفراء، جاروف ومكشطة).
الإجراء الفوري:
تأمين المنطقة ومنع مرور الآخرين.
ارتداء معدات الوقاية الشخصية الكاملة من المجموعة.
إزالة أي زجاج مكسور أو أدوات حادة باستخدام الجاروف والملقط ووضعها في حاوية الأدوات الحادة.
احتواء الانسكاب باستخدام المواد الماصة أو البودرة وتركها للفترة الموصى بها لتتصلب (خاصة للدم).
إزالة المادة المتصلبة باستخدام الجاروف والمكشطة ووضعها في الكيس الأصفر.
تطهير المنطقة الملوثة جيدًا بالمطهر المناسب (مثل محلول الكلور المخفف 1:10 للدم) مع تركه للمدة المطلوبة (وقت التلامس).
مسح المطهر بقطعة قماش نظيفة والتخلص منها في الكيس الأصفر.
التخلص من جميع الأدوات المستخدمة ومعدات الوقاية الشخصية في الكيس الأصفر وإغلاقه بإحكام.
غسل اليدين جيدًا بالماء والصابون.
مكافحة الحشرات والقوارض: تطبيق برنامج دوري ومنتظم لمكافحة الآفات بالتعاون مع شركات متخصصة ومعتمدة، مع التركيز على مناطق المطبخ والتخزين والنفايات.
4.7 التعامل الآمن مع الأدوات والمعدات (Safe Handling of Instruments/Equipment)
الأهمية: منع انتقال العدوى من خلال الأدوات والمعدات الملوثة.
الإجراءات التنفيذية:
4.7.1 تصنيف سبولدينج (Spaulding Classification):
أدوات حرجة (Critical): تخترق الأنسجة المعقمة أو تدخل الأوعية الدموية (مثل أدوات الجراحة البسيطة إن وجدت، الإبر). تتطلب تعقيمًا (Sterilization).
أدوات شبه حرجة (Semi-critical): تلامس الأغشية المخاطية أو الجلد غير السليم (مثل بعض معدات العلاج التنفسي، مناظير المستقيم/المهبل إن وجدت). تتطلب تطهيرًا عالي المستوى (High-Level Disinfection - HLD) كحد أدنى.
أدوات غير حرجة (Non-critical): تلامس الجلد السليم فقط (مثل السماعة، جهاز الضغط، العكازات). تتطلب تطهيرًا منخفض أو متوسط المستوى.
4.7.2 الأدوات ذات الاستخدام الواحد (Single-Use Devices - SUDs):
السياسة: يمنع منعًا باتًا إعادة معالجة أو إعادة استخدام أي أداة مصنفة على أنها ذات استخدام واحد، حتى لو كانت لنفس المستفيد.
التخلص: يجب التخلص منها فورًا بعد الاستخدام في حاويات النفايات المناسبة (عادةً الحادة أو المعدية).
4.7.3 الأدوات القابلة لإعادة الاستخدام:
المعالجة عند نقطة الاستخدام: إزالة الأوساخ والمواد العضوية الظاهرة من الأداة مباشرة بعد استخدامها بمسحها بقطعة قماش مبللة بالماء أو بمنظف إنزيمي (حسب نوع الأداة).
النقل الآمن: نقل الأدوات الملوثة فورًا إلى منطقة التنظيف المخصصة أو قسم التعقيم المركزي (CSSD) إن وجد، داخل حاويات مغلقة، مقاومة للتسرب والثقب، وتحمل علامة الخطر البيولوجي. رش الأدوات بمادة حافظة (transport gel/spray) إذا كان النقل سيتأخر.
التنظيف الشامل: أهم خطوة. يجب تنظيف الأدوات جيدًا يدويًا (باستخدام فرشاة ومحلول منظف إنزيمي مناسب تحت سطح الماء لمنع تناثر الرذاذ) أو آليًا (في غسالة مطهرة) لإزالة جميع البقايا العضوية وغير العضوية.
التطهير/التعقيم:
الأدوات شبه الحرجة: تخضع للتطهير عالي المستوى (HLD) باستخدام مواد كيميائية معتمدة (مثل الجلوترالدهيد، بيروكسيد الهيدروجين) مع الالتزام التام بالتركيز ووقت التلامس ودرجة الحرارة الموصى بها، ثم تشطف بماء معقم وتجفف جيدًا.
الأدوات الحرجة: تخضع للتعقيم باستخدام طرق معتمدة (مثل الأوتوكلاف بالبخار للأدوات المقاومة للحرارة والرطوبة، أو طرق أخرى للأدوات الحساسة) مع مراقبة دقيقة لدورة التعقيم باستخدام المؤشرات الفيزيائية والكيميائية والبيولوجية.
التغليف والتخزين: تغليف الأدوات المعقمة بشكل مناسب للحفاظ على تعقيمها، وتخزينها في منطقة نظيفة وجافة ومخصصة، مع تدوين تاريخ التعقيم وتاريخ انتهاء الصلاحية (حسب سياسة المركز). تطبيق مبدأ "ما يدخل أولاً يخرج أولاً" (FIFO).
4.8 النظافة التنفسية وآداب السعال (Respiratory Hygiene/Cough Etiquette)
الأهمية: الحد من انتشار الممرضات التي تنتقل عبر الرذاذ التنفسي.
الإجراءات التنفيذية:
التغطية: يجب على جميع الأفراد (العاملين، المستفيدين القادرين، الزوار) تغطية الفم والأنف بمنديل ورقي عند السعال أو العطس.
التخلص: التخلص من المنديل المستخدم فورًا في سلة مهملات (يفضل أن تكون تعمل بدون لمس).
البديل: في حالة عدم توفر منديل، يتم السعال أو العطس في أعلى الكم أو المرفق، وليس في اليدين.
نظافة الأيدي: غسل اليدين بالماء والصابون أو تطهيرها بالكحول بعد ملامسة الإفرازات التنفسية أو الأدوات الملوثة بها.
الكمامة للمرضى: تشجيع المستفيدين الذين يعانون من أعراض تنفسية على ارتداء كمامة جراحية عند التواجد بالقرب من الآخرين أو عند الخروج من غرفهم.
التوعية والمستلزمات:
وضع لوحات إرشادية واضحة (باللغتين) في مناطق الانتظار والمداخل والممرات للتوعية بأهمية آداب السعال.
توفير المناديل الورقية وسلال المهملات ومطهرات الأيدي الكحولية بسهولة في هذه المناطق.
4.9 ممارسات الحقن الآمن والتقنية التعقيمية (Safe Injection Practices & Aseptic Technique)
الأهمية: منع انتقال العدوى (خاصة المنقولة بالدم) المرتبطة بإجراءات الحقن والإجراءات التداخلية الأخرى.
الإجراءات التنفيذية:
4.9.1 مبادئ الحقن الآمن:
التقنية التعقيمية: الالتزام الصارم بالتقنية التعقيمية عند تحضير وإعطاء الحقن وجميع الإجراءات التداخلية (سيتم تفصيلها).
أدوات معقمة ذات استخدام واحد: استخدام إبرة ومحقنة معقمة جديدة لكل عملية حقن ولكل دخول لقارورة دواء (حتى لو كانت متعددة الجرعات).
قوارير الدواء:
ذات الجرعة الواحدة (Single-dose vials): تستخدم لمريض واحد وإجراء واحد فقط. يجب التخلص من أي كمية متبقية.
متعددة الجرعات (Multi-dose vials): إذا كان لا بد من استخدامها، يجب تخصيصها لمريض واحد كلما أمكن ذلك، مع تدوين اسم المريض وتاريخ الفتح عليها. إذا استخدمت لأكثر من مريض، يجب الاحتفاظ بها والتعامل معها فقط في منطقة تحضير الدواء النظيفة، وتطهير الغطاء المطاطي بالكحول قبل كل استخدام، والتخلص منها بعد 28 يومًا من الفتح أو حسب تاريخ انتهاء الصلاحية أو توصيات الشركة المصنعة (أيهما أقرب).
منطقة تحضير نظيفة: تحضير جميع الحقن في منطقة نظيفة ومخصصة بعيدًا عن مناطق التلوث المحتمل.
4.9.2 التعامل مع الإبر والأدوات الحادة (منع الوخز بالإبر NSI):
عدم إعادة التغطية: يمنع منعًا باتًا إعادة تغطية الإبر المستخدمة بكلتا اليدين.
تقنية السكوب (Scoop Technique): إذا كان لا بد من إعادة التغطية لسبب ما (وهو أمر غير مفضل)، يتم استخدام تقنية اليد الواحدة (تقنية السكوب).
عدم الثني أو الكسر: يمنع ثني أو كسر الإبر المستخدمة.
التخلص الفوري: التخلص من الإبر والمحاقن والأدوات الحادة الأخرى المستخدمة فورًا بعد الاستخدام مباشرة في حاوية الأدوات الحادة المخصصة والمقاومة للثقب والموجودة عند نقطة الاستخدام.
حاويات الأدوات الحادة: التأكد من توفرها، وضعها في مكان آمن ومناسب (الارتفاع الصحيح)، عدم ملئها لأكثر من ¾، وإغلاقها بإحكام قبل التخلص منها كنفايات طبية.
الأجهزة الآمنة: استخدام أدوات الحقن ذات آليات الحماية المدمجة (مثل الإبر ذاتية التغليف أو القابلة للسحب) كلما كان ذلك متاحًا.
4.9.3 التقنية التعقيمية في الإجراءات المختلفة:
المبدأ العام: الحفاظ على تعقيم الأدوات والمستلزمات والمحاليل المستخدمة في الإجراءات التي تتطلب تعقيمًا، ومنع تلوث الحقل المعقم.
نظافة الأيدي: غسل الأيدي الجراحي أو تطهيرها قبل الإجراءات المعقمة.
PPE: ارتداء القفازات المعقمة، وقد يتطلب الأمر ارتداء مئزر وقناع وواقي عين معقم حسب نوع الإجراء (مثل إدخال قسطرة وريدية مركزية).
تطهير جلد المريض: تطهير جلد المريض في موقع الإجراء باستخدام مطهر معتمد (مثل الكلورهيكسيدين الكحولي) وتركه ليجف تمامًا قبل البدء.
الحقل المعقم: استخدام أغطية (Drapes) معقمة لتحديد منطقة العمل والحفاظ عليها معقمة طوال الإجراء.
التعامل مع الأدوات المعقمة: التعامل مع الأدوات والمستلزمات المعقمة فقط بأيدٍ ترتدي قفازات معقمة أو باستخدام أدوات معقمة (مثل الملاقط). تجنب ملامسة الأدوات المعقمة للأسطح غير المعقمة.
المحاليل المعقمة: استخدام محاليل معقمة فقط، والتأكد من عدم تلوثها أثناء الاستخدام.
5. التدريب والتعليم المستمر
السياسة: يُعد الاستثمار في تدريب وتوعية جميع العاملين بشكل مستمر ومتخصص عنصرًا حاسمًا لضمان تطبيق سياسات مكافحة العدوى بفعالية وتحقيق بيئة آمنة ومستدامة في المركز.
الإجراءات التنفيذية:
5.1 أهمية التدريب
بناء الكفاءة: تزويد العاملين بالمعرفة والمهارات اللازمة لتطبيق الممارسات الصحيحة.
توحيد الممارسات: ضمان فهم وتطبيق الإجراءات بشكل متسق من قبل الجميع.
تعزيز الالتزام: زيادة الوعي بأهمية مكافحة العدوى وتحفيز العاملين على الالتزام.
مواكبة المستجدات: إطلاع العاملين على أحدث التطورات العلمية والإرشادات الوطنية.
تقليل الأخطاء: الحد من فرص حدوث الأخطاء أو الإهمال في تطبيق الإجراءات الوقائية.
تحسين النتائج: المساهمة في تقليل معدلات العدوى وتحسين سلامة المستفيدين والعاملين.
5.2 رخصة المهارات الأساسية لمكافحة العدوى (BICSL - "بيكسل")
5.2.1 الإلزامية والمحتوى:
هذا البرنامج إلزامي لجميع الممارسين الصحيين والعاملين الذين لهم اتصال مباشر أو غير مباشر بالمستفيدين أو بيئتهم في مركز التأهيل الشامل بالباحة.
يغطي البرنامج (7) مكونات أساسية وحيوية:
نظافة وتطهير الأيدي (HH).
استخدام معدات الوقاية الشخصية (PPE).
التعامل مع الانسكابات البيولوجية (Spill Kit Management).
التعامل مع وخز الإبر والإصابات الحادة (NSI Management).
تطبيق احتياطات العزل (Transmission-Based Precautions).
إجراء اختبار ملائمة الكمامة التنفسية (Respirator Fit Testing).
استخدام جهاز تنقية الهواء المزود بالطاقة (PAPR) عند الحاجة.
5.2.2 الترخيص والتجديد والمدربين المعتمدين:
يجب على جميع العاملين المستهدفين اجتياز التقييم النظري والعملي للحصول على رخصة "بيكسل".
مدة صلاحية الرخصة سنتان (وفقًا لأحدث التحديثات)، ويجب تجديدها قبل انتهاء صلاحيتها من خلال إعادة التدريب والتقييم.
يجب أن تكون بطاقة الرخصة سارية المفعول، مطبوعة، ومعلقة بشكل واضح على زي العامل أثناء ساعات العمل.
يتم تقديم التدريب والتقييم العملي من قبل مدربين معتمدين حاصلين على شهادة "مدرب بيكسل" (BICSL Trainer) من الجهات المعتمدة (مثل الإدارة العامة لمكافحة عدوى المنشآت الصحية بوزارة الصحة أو من تفوضه). يجب أن يكون لدى المركز عدد كافٍ من المدربين المعتمدين لتغطية احتياجات التدريب والتجديد لجميع العاملين.
5.3 التدريب التوجيهي للموظفين الجدد
يجب أن يتلقى جميع الموظفين الجدد (بما في ذلك العاملين الصحيين، الإداريين، مقدمي الرعاية، عمال النظافة، عمال المطبخ والمغسلة، والمتعاقدين) تدريبًا توجيهيًا شاملاً على أساسيات مكافحة العدوى وسياسات المركز المحددة قبل أو خلال الشهر الأول من بدء العمل.
يجب أن يشمل هذا التدريب على الأقل: أهمية مكافحة العدوى، سلسلة العدوى، الاحتياطات القياسية (خاصة نظافة الأيدي وآداب السعال)، مقدمة عن PPE، وإدارة النفايات الأساسية.
5.4 التدريب المستمر والمتخصص
التدريب السنوي الإلزامي: يجب عقد دورات تدريبية تنشيطية سنوية على الأقل لجميع العاملين لمراجعة المبادئ الأساسية، ومناقشة أي تحديثات في السياسات أو الإجراءات، ومراجعة نتائج المراقبة والتدقيق الخاصة بالمركز.
التدريب المبني على الاحتياج: تصميم وتقديم برامج تدريبية إضافية بناءً على:
نتائج تقييم المخاطر السنوي.
نتائج المراقبة والتدقيق التي تظهر ضعفًا في مجال معين.
ظهور أمراض معدية جديدة أو مستجدة.
إدخال تقنيات أو معدات جديدة تتطلب ممارسات مكافحة عدوى خاصة.
التدريب المتخصص: توفير تدريب معمق ومفصل للفئات التي تتطلب مهارات متخصصة، مثل:
العاملون في المطبخ: سلامة الأغذية، النظافة الشخصية، تنظيف وتطهير معدات المطبخ.
العاملون في المغسلة: التعامل الآمن مع المنسوجات الملوثة، إجراءات الغسيل والتطهير الصحيحة.
العاملون في النظافة: تقنيات التنظيف والتطهير الصحيحة، التعامل مع المطهرات والمواد الكيميائية، تنظيف غرف العزل.
العاملون في إدارة النفايات: إجراءات الفرز والجمع والنقل والتخزين الآمن.
الفريق الطبي والتمريضي: تطبيق حزم الرعاية (Bundles)، التقنية التعقيمية للإجراءات التداخلية، التعامل مع الأجهزة الطبية الخاصة.
5.5 تقييم الكفاءة والتوثيق
التقييم: يجب ألا يقتصر التدريب على تقديم المعلومات، بل يجب أن يتضمن تقييمًا لكفاءة المتدربين في تطبيق المهارات والمعارف المكتسبة. يمكن أن يتم التقييم من خلال:
الاختبارات النظرية: لقياس مستوى المعرفة والفهم.
الملاحظة المباشرة للأداء: تقييم تطبيق المهارات عمليًا (مثل تقنية غسل اليدين، ارتداء وخلع PPE).
المحاكاة (Simulation): استخدام سيناريوهات محاكاة لتقييم الاستجابة في مواقف معينة.
أدوات تقييم BICSL: استخدام أدوات التقييم المعتمدة مثل نموذج التقييم العملي للمدرب (F2023A) ونموذج التقييم الذاتي للتدريب (SA2023A) كجزء من عملية التقييم والجودة لبرنامج "بيكسل".
التوثيق: يجب توثيق جميع الأنشطة التدريبية بدقة في سجلات خاصة أو في ملفات الموظفين، ويشمل التوثيق:
عنوان الدورة التدريبية وتاريخها ومدتها.
أسماء الحضور وتوقيعاتهم.
اسم المدرب.
نتائج تقييم الكفاءة (درجات الاختبارات، نتائج الملاحظة).
تاريخ صلاحية الشهادات (مثل BICSL) ومواعيد التجديد المطلوبة.
6. الصحة المهنية للعاملين
السياسة: يلتزم مركز التأهيل الشامل بالباحة بتوفير بيئة عمل آمنة وصحية لجميع العاملين، وحمايتهم من المخاطر المهنية المرتبطة بالعدوى، وذلك من خلال تطبيق برنامج شامل ومتكامل للصحة المهنية وفقًا لمتطلبات وزارة الصحة واللوائح الوطنية.
الإجراءات التنفيذية:
6.1 أهمية حماية العاملين
العاملون الصحيون ومقدمو الرعاية هم خط الدفاع الأول للمستفيدين، وحمايتهم تضمن استمرارية وجودة الرعاية.
الوقاية من إصابة العاملين بالعدوى تمنع نقلها إلى المستفيدين الآخرين أو الزملاء أو أفراد أسرهم.
تقليل الغياب عن العمل بسبب المرض المهني يعزز كفاءة التشغيل في المركز.
توفير بيئة عمل آمنة يعزز معنويات العاملين ورضاهم الوظيفي.
6.2 متطلبات برنامج الصحة المهنية
6.2.1 الفحص والتقييم (قبل التوظيف ودوري):
الفحص الأولي: إجراء فحص طبي شامل لجميع الموظفين الجدد قبل مباشرة العمل، يتضمن تقييمًا للتاريخ الصحي وفحصًا سريريًا وفحوصات مخبرية أساسية للكشف عن الأمراض المعدية الهامة (مثل التهاب الكبد B و C، فيروس نقص المناعة البشرية HIV، والسل TB).
فحص السل (TB):
إجراء اختبار الجلد التوبركوليني (PPD/TST) لجميع العاملين الجدد (حتى لو تلقوا لقاح BCG سابقًا، إلا إذا كان لديهم توثيق لنتيجة إيجابية سابقة). يفضل إجراء اختبار الخطوتين (Two-step TST) إذا لم يكن هناك سجل لاختبار سلبي خلال الـ 12 شهرًا الماضية.
إعادة اختبار PPD سنويًا لجميع العاملين الذين كانت نتيجتهم سلبية في الفحص الأساسي.
مراقبة وحساب معدلات التحول الإيجابي (PPD Conversion Rate) سنويًا.
إحالة أي حالة إيجابية جديدة (أو حالة لديها أعراض توحي بالسل) لإجراء أشعة سينية للصدر وتقييم طبي كامل للنظر في العلاج الوقائي أو علاج المرض النشط.
6.2.2 التحصين والتطعيمات:
تقييم المناعة: تحديد الحالة المناعية لجميع العاملين الجدد ضد الأمراض التالية التي يمكن الوقاية منها باللقاحات:
التهاب الكبد B (Hepatitis B): من خلال سجل تطعيم موثق (3 جرعات) أو نتيجة تحليل الأجسام المضادة (Anti-HBs ≥ 10 mIU/mL).
الحصبة، النكاف، الحصبة الألمانية (MMR): من خلال سجل تطعيم موثق (جرعتان) أو دليل مصلي للمناعة (IgG إيجابي) أو تشخيص سريري/مخبري موثق للمرض.
جدري الماء (Varicella/Chickenpox): من خلال سجل تطعيم موثق (جرعتان) أو دليل مصلي للمناعة (IgG إيجابي) أو تشخيص سريري/مخبري موثق للمرض.
توفير اللقاحات: إعطاء اللقاحات اللازمة (HBV, MMR, Varicella) لجميع العاملين الذين يتبين أنهم غير محصنين أو لديهم مناعة غير كافية، وفقًا للجداول الزمنية المعتمدة.
لقاح الإنفلونزا الموسمية: تقديم وتوفير لقاح الإنفلونزا سنويًا لجميع العاملين في المركز وتشجيعهم بقوة على أخذه، وفقًا لتوصيات وزارة الصحة.
لقاحات أخرى: توفير لقاحات إضافية حسب الحاجة بناءً على تقييم المخاطر أو التوصيات الوطنية (مثل لقاح المكورات السحائية للعاملين في الحج أو المختبرات إن وجد).
6.2.3 إدارة التعرض للمخاطر (وخز الإبر، سوائل الجسم):
الإبلاغ الفوري: يجب على أي عامل يتعرض لوخز إبرة أو جرح بأداة حادة ملوثة، أو تعرض جلده غير السليم أو أغشيته المخاطية للدم أو سوائل الجسم، أن يقوم فورًا بالإبلاغ عن الحادثة لمشرفه المباشر وللقسم الطبي/لجنة مكافحة العدوى بالمركز.
الإسعافات الأولية:
الوخز/الجرح: غسل المنطقة فورًا وبلطف بالماء والصابون، ثم تطهيرها بمطهر مناسب (مثل الكحول 70%). لا تقم بعصر الجرح.
تعرض الأغشية المخاطية (العين، الفم، الأنف): شطف المنطقة فورًا بكمية وفيرة من الماء النظيف أو محلول الملح الفسيولوجي.
التقييم والمتابعة:
تقييم طبي فوري للعامل المعرض لتقييم خطورة التعرض.
تقييم مصدر التعرض (المستفيد) بعد أخذ الموافقة، للكشف عن وجود فيروسات التهاب الكبد B و C وفيروس نقص المناعة البشرية HIV.
تقديم العلاج الوقائي بعد التعرض (PEP) للعامل إذا لزم الأمر (مثل لقاح وجلوبيولين مناعي لالتهاب الكبد B، أو أدوية HIV) بناءً على تقييم المخاطر ووفقًا للبروتوكولات الوطنية.
إجراء الفحوصات المصلية اللازمة للعامل (أساسية ودورية للمتابعة) حسب نوع التعرض ونتائج المصدر.
تقديم المشورة والدعم النفسي للعامل المعرض.
الوقاية: تحليل بيانات حوادث التعرض لتحديد الأسباب (مثل الإجراءات عالية الخطورة، الأدوات غير الآمنة، عدم الالتزام بالممارسات الآمنة) وتطبيق إجراءات وقائية لمنع تكرارها (مثل التدريب، توفير حاويات الأدوات الحادة بشكل كافٍ، استخدام الأجهزة ذات آليات الحماية).
6.2.4 تقييد العمل (Work Restrictions):
السياسة: تطبيق سياسة واضحة ومعتمدة (تستند إلى البروتوكولات الوطنية/وزارة الصحة) لتقييد عمل الموظفين الذين يعانون من أمراض معدية يمكن أن تنتقل إلى المستفيدين أو الزملاء.
الأمراض التي تستدعي التقييد (أمثلة): الإسهال الحاد، التهابات الجهاز التنفسي الحادة المصحوبة بحمى، التهاب الملتحمة المعدي، الهربس النشط (خاصة على اليدين أو الوجه)، الآفات الجلدية المتقيحة أو النازة، السل الرئوي النشط، الحصبة، جدري الماء، النكاف، التهاب الكبد A الحاد.
الإجراء: يجب على الموظف الذي يعاني من أعراض توحي بمرض معدٍ الإبلاغ فورًا لمشرفه والقسم الطبي. يتم تقييم الحالة من قبل الطبيب لتحديد مدى الحاجة لتقييد العمل ومدته ونوعه (مثل منع التعامل المباشر مع المستفيدين، أو الإجازة المرضية، أو العزل).
العزل المنزلي للعاملين: في حال تطلب الأمر عزل العامل، يتم ذلك إما في منزله الخاص أو في غرف مخصصة في سكن الموظفين (إذا كان متوفرًا ومناسبًا) أو في غرف مخصصة بالمستشفى إذا لزم الأمر، مع تطبيق احتياطات العزل المناسبة.
العودة للعمل: لا يسمح للعامل بالعودة للعمل إلا بعد التأكد من زوال خطر نقل العدوى بناءً على تقييم طبي ووفقًا لمدة التقييد المحددة في السياسة.
6.2.5 السجلات الصحية:
الاحتفاظ بملف صحي سري ومحدث لكل عامل في المركز (بما في ذلك العاملين الصحيين والإداريين وعمال الخدمات المساندة والمتعاقدين).
يجب أن يتضمن الملف جميع بيانات الفحص الطبي قبل التوظيف والدوري، سجلات التطعيمات، نتائج اختبارات السل، أي حوادث تعرض مهني وإجراءات المتابعة، وأي فترات لتقييد العمل.
7. المراقبة والتقييم والتحسين المستمر
السياسة: يُعد التقييم المنتظم والمراجعة الدورية لجميع جوانب برنامج مكافحة العدوى أمرًا ضروريًا لضمان فعاليته، وتحديد نقاط القوة والضعف، وتطبيق التحسينات اللازمة بشكل مستمر وممنهج.
الإجراءات التنفيذية:
7.1 آليات المراقبة والتدقيق
المراقبة اليومية والمستمرة: من خلال الملاحظة المباشرة للممارسات واستخدام قوائم المراجعة اليومية/الأسبوعية من قبل المشرفين وممارسي مكافحة العدوى.
التدقيق الدوري المنهجي: إجراء عمليات تدقيق شاملة باستخدام أدوات موحدة (مثل معايير ICA أو أدوات التدقيق الوطنية) لتقييم الالتزام بالسياسات في جميع الأقسام بشكل ربع سنوي أو نصف سنوي، بتنظيم وإشراف لجنة مكافحة العدوى.
مراجعة السجلات: التدقيق في السجلات المختلفة (التدريب، التنظيف، الصيانة، النفايات، التعرض) كجزء من عملية التقييم.
7.2 مؤشرات الأداء الرئيسية (KPIs)
تحديد ومتابعة مؤشرات أداء واضحة وقابلة للقياس لتقييم فعالية البرنامج، مثل:
معدل الالتزام بنظافة الأيدي (%).
معدل الالتزام باستخدام PPE الصحيح (%).
معدل اكتمال التدريب الإلزامي (%).
معدل حوادث الوخز بالإبر لكل 100 موظف (أو لكل 1000 يوم عمل).
نسبة العاملين المحصنين ضد الإنفلونزا (%).
معدل دوران PPD (%).
معدلات العدوى المرتبطة بالرعاية الصحية (HAIs) (لكل 1000 يوم مريض أو 1000 يوم جهاز، إذا كان يتم ترصدها).
7.3 تحليل البيانات والإبلاغ
تجميع وتحليل بيانات المراقبة والتدقيق ومؤشرات الأداء بشكل دوري (شهري/ربع سنوي).
تحديد الاتجاهات، ومقارنة الأداء بالمعايير أو الأهداف المحددة.
إعداد تقارير واضحة وموجزة تلخص النتائج وتقدمها إلى لجنة مكافحة العدوى وإدارة المركز.
7.4 التغذية الراجعة والإجراءات التصحيحية
مناقشة النتائج: عرض ومناقشة تقارير المراقبة والتقييم في اجتماعات لجنة مكافحة العدوى.
تحديد الأولويات: تحديد مجالات التحسين ذات الأولوية بناءً على تحليل البيانات وتقييم المخاطر.
خطط العمل التصحيحية:
وضع خطط عمل محددة وقابلة للتنفيذ لمعالجة أوجه القصور أو المشكلات التي تم تحديدها.
يجب أن تتضمن الخطة: المشكلة، الإجراء التصحيحي المقترح، الشخص/القسم المسؤول عن التنفيذ، الجدول الزمني، ومؤشر قياس النجاح.
التغذية الراجعة: تقديم تغذية راجعة بناءة ومنتظمة (شفهية ومكتوبة) للأقسام والموظفين المعنيين حول نتائج التقييم وأدائهم، وتسليط الضوء على الممارسات الجيدة ومجالات التحسين.
المتابعة: متابعة تنفيذ خطط العمل التصحيحية وتقييم فعاليتها في تحقيق التحسين المطلوب وإغلاق حلقة الجودة.
7.5 التوثيق
توثيق جميع مراحل عملية المراجعة والتقييم، بما في ذلك التقارير، وخطط العمل التصحيحية، ونتائج المتابعة، في محاضر اجتماعات اللجنة أو سجلات خاصة بالبرنامج.
